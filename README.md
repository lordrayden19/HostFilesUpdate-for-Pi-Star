# HostFilesUpdate-for-Pi-Star
HostFilesUpdate for Pi Star agar dmr host dan dmrids.dat bisa diatur sesuai sumber yang diinginkan

💡 Tips Setelah Upload ke Pi-Star:
Jangan lupa untuk memberikan izin eksekusi (executable permission) pada file tersebut setelah di-upload ke sistem Pi-Star agar script bisa berjalan dengan perintah berikut lewat SSH Terminal:

Bash
sudo chmod +x /usr/local/sbin/HostFilesUpdate.sh
(Catatan: Sesuaikan jalur lokasi /usr/local/sbin/ jika Anda meletakkannya di folder lain).
